# AI Sniper (production-ready)

This repository contains a production-ready Flask + background scanner for M1 signals.
Files:
- main.py (app + scanner)
- requirements.txt
- Procfile
- .env.example

Quick setup (GitHub web UI)
1. Create a new repository named `ai-telegram-bot` under your account `MehediHasan`.
2. Add the files above (use "Add file" -> "Create new file" and paste contents).
3. Commit to `main` branch.

Or local git:
```bash

git branch -M main
# replace <your-remote-url> with the repo URL
